# HumanPose-Tracker-using-MediaPipe-OpenCV
PoseTrack AI – Real-Time Human Pose Detection
## 📌 Project Overview

PoseTrack AI is a real-time human pose detection system built using Python, OpenCV, and MediaPipe BlazePose.
The application captures video from a webcam and detects human body landmarks, drawing a skeletal structure over the person in real time.

This project demonstrates how AI-powered computer vision can track human body movements efficiently using lightweight models.

## 🎯 Features

🔴 Real-time pose detection using webcam

🧍 Detects 33 body landmarks

📍 Tracks key joints like shoulders, elbows, knees, and ankles

⚡ Fast and lightweight using MediaPipe BlazePose

🖥️ Live skeletal visualization on video feed

🧠 Beginner-friendly computer vision project

## 🧠 Technologies Used

## 🏗️ Project Workflow

1️⃣ Capture video frames using OpenCV

2️⃣ Convert frame from BGR → RGB

3️⃣ Process frame using MediaPipe BlazePose

4️⃣ Detect body landmarks

5️⃣ Draw pose skeleton on frame

6️⃣ Display output in real time
